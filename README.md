# Social Science Replication Project

## What is the Project about?

This project is about replicating the paper Good News Is Not a Sufficient Condition for Motivated Reasoning [https://arxiv.org/abs/2012.01548] by Michael Thaler with the public available data find at the Harvard Dataverse [https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/58OI0B]. The project is part of a university task and not related to the initial author.
